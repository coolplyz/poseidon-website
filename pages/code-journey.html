    Day 6 of #100DaysofCode

    Learnings:

    • box model
    • everything has a box around it
    • margin: space between border of box and adjacent boxes
    • padding: space between border of box and content of box
    • borders: space between margin and padding
    • content box(standard box model): padding and borders are added to the actual pixel size -> content will not change
    • border box(alternate box model): padding and borders will be included in total size -> content will shrink
    • display tpes
    • outer display types: block and inline
    • inner display types: flex or grid
